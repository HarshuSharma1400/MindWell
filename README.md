MindWell – Mental Wellness Web Application

📌 Project Overview


MindWell is a full-stack web application developed as part of an MCA final year project. The system is designed to support users in maintaining and improving their mental well-being through a combination of self-assessment tools, guided wellness content, and personal tracking features.


The platform provides a safe and user-friendly environment where users can monitor their emotional health, practice mindfulness, and access helpful mental health resources in a structured way.


🎯 Key Features

🧠 PHQ-9 Mental Health Assessment

Allows users to evaluate their depression severity using a standardized questionnaire.

📓 Personal Journal

Enables users to write and manage daily thoughts, helping in emotional expression and reflection.

🧘 Meditation & Mindfulness

Provides guided content to reduce stress and improve focus.

📊 Mood Tracking & Visualization

Helps users track their mental health trends using graphical representations.

🔐 User Authentication System

Secure registration and login functionality with session management.

🎧 Calm Audio & Stories

Offers relaxing audio content to support stress relief and mental relaxation.

💪 Wellness Exercises

Includes simple exercises that promote physical and mental well-being.

⚙️ System Setup & Installation

🔹 Prerequisites


Ensure the following tools are installed:


Node.js (v14 or above)

MongoDB (Local or Atlas)

Any API testing tool (e.g., Postman)

🔹 Backend Setup

cd server

npm install

Create a .env file:


PORT=3000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key


Run the backend server:


npm run server


🔹 Frontend Setup

cd client

npm install

npm run dev
